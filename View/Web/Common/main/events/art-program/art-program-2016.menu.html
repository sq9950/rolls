光影交织，独具匠心

自2014年劳斯莱斯汽车艺术项目全面启动以来，劳斯莱斯汽车开始了与来自世界各地艺术家们的深度合作，劳斯莱斯汽车此次力邀中国影像艺术家杨福东先生加盟劳斯莱斯汽车艺术项目，合作推出了艺术短片《愚公移山》（MOVING MOUNTAINS）。

这部2016年创作的影像作品，采用当代艺术形式，重新阐述了《愚公移山》这则有关毅力和信念的古老故事。杨福东先生将这则中国古老故事与现代精神融合、碰撞，借此展开对人性及不断变化的价值观的诗意反思，并带出了影片另一核心主题：母亲的精神。作品已在11月完成，并于2016年12月13日至2017年1月5日期间，在位于上海西岸文化走廊的上海摄影艺术中心公开展出！

恒久之念 契而不舍

这部时长46分钟的黑白影像作品，灵感源自国画大师徐悲鸿先生在上世纪40年代创作的巨作《愚公移山》，杨福东先生在创作时融入其个人特色的单色基调，展现出作品回归自然主义的标志性特点，借愚公在自我封闭世界中的境遇，引发人类无休止的劳作随之带来的环境问题等思考。

杨福东先生入世而不随波逐流的创作思想，同劳斯莱斯汽车尊重传统却又不断创新的品牌理念不谋而合。影片《愚公移山》在表达坚持不懈、锲而不舍的中国传统文化精髓的同时，生动呈现了勇于创新开拓的时代精神——这正是劳斯莱斯汽车所倡导和实践的。

艺术之恋 天作之合

杨福东先生生于1971年，是中国当代影响力深远的艺术家之一。在合作伊始，劳斯莱斯汽车邀请了杨福东先生专程造访位于英国古德伍德的总部，旨在通过让这位艺术家现场感受劳斯莱斯的品牌精髓，并为其新作品的创作提供灵感。“对于劳斯莱斯汽车来说，不论是工艺设计，还是品质性能，有很多东西是值得赞赏的。”杨福东先生说，“每一台劳斯莱斯汽车的问世都是从极简到极致的过程，它不亚于一件艺术品的诞生。在劳斯莱斯汽车身上有很多质感的元素，在我看来，它和艺术是相通的。”

上海摄影艺术中心创始人刘香成先生表示，“劳斯莱斯汽车有着独特的设计与气质，它与艺术的结合就好比是一场自然而然的婚姻，在我眼里，这就是天作之合。”

传世工艺 经典融合

在上百年的荣耀历史中，劳斯莱斯汽车的设计与品牌基因始终与艺术密不可分，在遵循艺术美学理念的同时，不断开创并引导汽车美学的风潮。每一辆劳斯莱斯汽车都是经过Bespoke客户定制业务，精选优质的材料，呈现有着上乘美学的经典艺术珍品。

劳斯莱斯汽车艺术项目，正是连接这个超豪华汽车品牌和艺术界的纽带——激发具有创造力的艺术作品，鼓励艺术家开启新的项目，并激励他们不断探索新的领域！
