我们合作的艺术项目

每一件劳斯莱斯汽车都是稀有、独特且富有美感的汽车艺术品。劳斯莱斯汽车艺术项目正式连接这个超豪华汽车品牌和艺术界的纽带，项目包括艺术作品的全球巡展，与全球知名组织的合作、与世界闻名的艺术家展开的艺术对话、艺术盛会、艺术漫步和艺术接待。
