Mysore古思特加长特别版

这款定制车型名称取自“迈索尔之虎”，即印度南部迈索尔王国的传奇式苏丹和统治者，车辆将老虎的力量和孔雀的英勇精妙融合。手工绘制的孔雀令双腰线更显优雅，与孔雀蓝和白色双色调外观相得益彰，为古思特再添璀璨之美。

在Mysore古思特加长特别版内部，嵌有与外部相同的主题图案。头枕带有手工缝制的老虎图案，进一步凸显了贝壳色真皮的奢华。放下中央扶手后，精巧设计呼之欲出：鹿皮色和鲜艳松石绿色羽毛刺绣犹如孔雀开屏般艳丽夺目。

Suhail Collection 船帆星座定制车型

Suhail Collection 船帆星座定制车型专为纪念数学家和天文学家伊本·阿尔·海塞姆对世界影响深远的一生。车型名称源自从阿拉伯半岛能够仰望的星星序列，月长石珍珠外观形如其名，熠熠生辉。在车内，浅奶油色真皮、手工制作的星星图案以及可定制星光顶饰将您带入另一个世界，徜徉在属于您自己的星座之下。

古思特成都熊猫特别版

这款定制版古思特风采绝伦，气魄非凡，旨在向中国最珍贵的居民致敬。青铜色和钯金色圆滑外观散发出沉稳从容的气质。深棕色木饰面板和柔软贝壳色真皮形成对比，再搭配明翠绿装饰，让您倍感舒适。

熊猫图案经手工刺绣缝入头枕。从腰线到新几内亚胡桃木横纹仪表盘，再到野餐桌，竹质细节设计遍布全车。这款客户定制车型展示了每辆劳斯莱斯的杰出工艺，在纷繁世界中为您营造一片宁静绿洲。

古思特英骥典藏版

这是半个多世纪以来劳斯莱斯初次将中国马年与木质元素集于一身。古思特英骥典藏版对中国艺术中的骏马以现代风格进行重新诠释，以示庆祝。

骏马主题融入双腰线设计，并手工缝入头枕和座垫，构成一个精致的主题元素。更加令人印象深刻的是嵌入橡木仪表盘的骏马图案。主题图案采用美国胡桃木、带状条纹红影木、枫木和暗色烟熏橡木打造，是客户定制服务团队的又一非凡杰作。

古思特Bold Pink特别版

古思特Bold Pink特别版由一对企业家夫妇委托定制，意在庆祝他们十周年结婚纪念，是活力设计的典型代表。夫妻二人的灵感来自长期以来对劳斯莱斯艺术设计的赞赏，这款定制作品不仅仅是一件视觉上的杰作，他们认为这更是一件礼物，让他们共同打造未来的回忆。古思特Bold Pink特别版采用明亮柔和的色调，不仅引人注目，同时也标志着一个终生目标的完成。
