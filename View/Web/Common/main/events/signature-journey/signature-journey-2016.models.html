传奇风范 首秀中国

2016年7月22日，劳斯莱斯汽车“风范尊行·无畏之旅”极致赛道体验活动在上海国际赛车场正式拉开帷幕。除了在赛道上展示劳斯莱斯汽车的速度、动力及优雅格调，此次“风范尊行·无畏之旅”赛道日还开启了超豪华汽车领域中散发强烈非凡气质的全新车型——魅影Black Badge车型的中国首秀。

此次劳斯莱斯汽车“风范尊行·无畏之旅”赛道日中国首秀的魅影Black Badge车型是劳斯莱斯汽车打造的动感而敏捷的车型。无论从工程技术还是美学理念来说，魅影Black Badge车型展现出劳斯莱斯汽车更为神秘、果敢、自信以及力臻完美 (Striving for perfection) 的另一面。

在劳斯莱斯汽车“风范尊行·无畏之旅”极致赛道体验活动上海站，来宾们除了能够品鉴劳斯莱斯Black Badge的优雅格调，还可以在F1赛道上尽情地感受劳斯莱斯魅影和古思特车型杰出的动力传动系统和操控系统，享受它们的舒适和力量。

无畏之旅 圆满收官

劳斯莱斯汽车“风范尊行·无畏之旅”在中国完成了宁波、郑州、长沙、深圳、广州等10个城市的巡回亮相，于2016年10月21日在珠海国际赛车场为客户及媒体提供了劳斯莱斯古思特Black Badge和魅影Black Badge在中国的首次赛道体验。劳斯莱斯汽车的Black Badge车型系列呈现给大家的是诠释动感奢华且独具个性，提供非凡驾乘体验的劳斯莱斯。

劳斯莱斯汽车不仅是精湛手工工艺的典范，也是车辆工程学的集大成之作。秉承着两位创始人查尔斯·劳斯以及亨利·莱斯的开拓精神与力臻完美 (Striving for perfection) 的态度，劳斯莱斯汽车一直以来从未放弃对激越速度、出色工程设计与舒适驾驶体验的不懈追求。继去年劳斯莱斯汽车首次举办“风范尊行”赛道日之后，今年，劳斯莱斯汽车再次将这样的精神品质在赛道上诠释无遗。

在劳斯莱斯汽车“风范尊行·无畏之旅”极致赛道体验活动珠海站，来宾们在赛道上试驾体验了劳斯莱斯Black Badge车型，充分领略超豪华汽车全新高级定制车型的不凡魅力，唤起在场来宾不循规蹈矩、恣意洒脱、迎难而上、力求非凡的情感共鸣，为2016年劳斯莱斯汽车“风范尊行·无畏之旅”画上圆满句号。
